# text_classification_system(文本分类项目)
文本分类系统，覆盖传统机器学习、深度学习到最前沿的预训练模型的全栈技术栈，并深入模型压缩部署优化;
- 技术栈覆盖：贯穿特征工程→经典算法→前沿模型→模型压缩全流程
- 优化：特别包含量化/蒸馏/剪枝等部署必备压缩技术
- 多方案对比：同一数据集下对比随机森林/FastText/BERT性能差异.
